**melcom's FFmpeg Audio Normalizer v2.2.1 (GUI Edition)**

Mit diesem Programm kannst Du die Lautstärke Deiner Audiodateien einfach mit FFmpeg normalisieren.
Die normalisierte Datei wird automatisch im gleichen Ordner wie die Originaldatei gespeichert.

**Kurzanleitung für Dich:**

1. **FFmpeg installieren:**
   - Für dieses Programm benötigst Du das kostenlose Programm `ffmpeg.exe`.
   - Lade `ffmpeg.exe` von einer vertrauenswürdigen Webseite herunter, zum Beispiel von https://github.com/BtbN/FFmpeg-Builds/releases/tag/latest.
   - Platziere den Ordner, der `ffmpeg.exe` enthält, auf Deinem Computer.

2. **Programm starten und FFmpeg Pfad angeben:**
   - Starte `AudioNormalizer.exe`.
   - Gehe im Programm zu "Datei" -> "Optionen".
   - Gib unter "FFmpeg Pfad" den Ordner an, in dem sich `ffmpeg.exe` befindet. Dies ist wichtig, damit das Programm FFmpeg nutzen kann.

3. **Audiodatei normalisieren:**
   - Klicke im Hauptfenster auf "Durchsuchen" und wähle die Audiodatei aus, die Du normalisieren möchtest.
   - Wähle unter "LUFS Preset" eine Voreinstellung für die Lautstärke (z.B. "YouTube" oder "Spotify") oder gib eigene Werte ein.
   - Wähle unter "Ausgabeformat" das gewünschte Format für die normalisierte Datei (z.B. WAV, MP3).
   - Klicke auf "Normalisierung starten".
   - Der Fortschritt wird angezeigt, und die normalisierte Datei wird automatisch erstellt.

4. **Optionen (Einstellungen):**
   - Im Menü "Datei" -> "Optionen" kannst Du den Pfad zu FFmpeg ändern, die Programmsprache auswählen und Logdatei-Einstellungen anpassen.

5. **Detaillierte Hilfe:**
   - Für ausführlichere Informationen öffne die Hilfedateien im Ordner `help`.
   - Öffne `help_de_DE.html` (für Deutsch), `help_en_US.html` (für Englisch) oder `help_pl_PL.html` (für Polnisch) mit Deinem Webbrowser.

**Wichtige Hinweise für Dich:**

* **FFmpeg Pfad:** Stelle sicher, dass der Pfad zu FFmpeg in den Optionen korrekt angegeben ist, sonst funktioniert das Programm nicht.
* **LUFS Presets:** Die Voreinstellungen helfen Dir, die richtige Lautstärke für verschiedene Plattformen zu wählen (z.B. YouTube, Spotify). Du kannst aber auch eigene Werte eingeben.
* **Ausgabeformat:** Wähle das gewünschte Audioformat für die normalisierte Datei. WAV ist verlustfrei und für höchste Qualität empfohlen.
