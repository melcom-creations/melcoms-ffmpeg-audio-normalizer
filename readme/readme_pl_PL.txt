**melcom's FFmpeg Audio Normalizer v2.2.1 (GUI Edition)**

Za pomocą tego programu możesz łatwo normalizować głośność swoich plików audio za pomocą FFmpeg.
Znormalizowany plik jest automatycznie zapisywany w tym samym folderze, co plik oryginalny.

**Szybki przewodnik dla Ciebie:**

1. **Zainstaluj FFmpeg:**
   - Do tego programu potrzebujesz darmowego programu `ffmpeg.exe`.
   - Pobierz `ffmpeg.exe` z zaufanej strony internetowej, na przykład z https://github.com/BtbN/FFmpeg-Builds/releases/tag/latest.
   - Umieść folder zawierający `ffmpeg.exe` na swoim komputerze.

2. **Uruchom program i wskaż ścieżkę FFmpeg:**
   - Uruchom `AudioNormalizer.exe`.
   - W programie przejdź do "Plik" -> "Opcje".
   - W polu "Ścieżka FFmpeg" wskaż folder, w którym znajduje się `ffmpeg.exe`. Jest to ważne, aby program mógł korzystać z FFmpeg.

3. **Znormalizuj plik audio:**
   - W oknie głównym kliknij "Przeglądaj" i wybierz plik audio, który chcesz znormalizować.
   - W polu "Preset LUFS" wybierz preset głośności (np. "YouTube" lub "Spotify") lub wprowadź własne wartości.
   - W polu "Format wyjściowy" wybierz żądany format dla znormalizowanego pliku (np. WAV, MP3).
   - Kliknij "Start Normalizacji".
   - Postęp będzie wyświetlany, a znormalizowany plik zostanie utworzony automatycznie.

4. **Opcje (Ustawienia):**
   - W menu "Plik" -> "Opcje" możesz zmienić ścieżkę do FFmpeg, wybrać język programu i dostosować ustawienia pliku logu.

5. **Szczegółowa Pomoc:**
   - Aby uzyskać bardziej szczegółowe informacje, otwórz pliki pomocy w folderze `help`.
   - Otwórz `help_de_DE.html` (dla języka niemieckiego), `help_en_US.html` (dla języka angielskiego) lub `help_pl_PL.html` (dla języka polskiego) za pomocą przeglądarki internetowej.

**Ważne Wskazówki dla Ciebie:**

* **Ścieżka FFmpeg:** Upewnij się, że ścieżka do FFmpeg jest poprawnie określona w opcjach, w przeciwnym razie program nie będzie działać.
* **Presety LUFS:** Presety pomagają Ci wybrać odpowiednią głośność dla różnych platform (np. YouTube, Spotify). Możesz jednak również wprowadzić własne wartości.
* **Format wyjściowy:** Wybierz żądany format audio dla znormalizowanego pliku. WAV jest bezstratny i zalecany dla najwyższej jakości.
