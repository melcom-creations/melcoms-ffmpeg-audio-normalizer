Flickwerk Edition - Modding-Anleitung: Eigene Themes & Sprachen

Teil 1: Ganz einfach - Sprache hinzufügen

Diese Anleitung ist für alle, die Sprachen selbst anpassen wollen. Du brauchst kein Python und keine Programmierkenntnisse. Ein normaler Texteditor reicht aus. Damit kannst du vorhandene Sprachdateien ändern oder neue Sprachdateien erstellen.

Keine Programmierkenntnisse nötig

Öffne einfach eine Sprachdatei im Ordner lang mit einem normalen Texteditor. Du kannst sie bearbeiten, kopieren oder neu anlegen - mehr brauchst du nicht.

Damit kannst du vorhandene Sprachen ändern oder eine neue Sprache hinzufügen. Das klappt ganz ohne Quellcode.

Schritt 1: Eine Datei kopieren

Öffne den Ordner lang und kopiere eine vorhandene Sprachdatei. Benenne die Kopie zum Beispiel in fr_FR.json, es_ES.json, it_IT.json, nl_NL.json oder pt_BR.json um.

Schritt 2: Nur den Text rechts ändern

Ganz wichtig: Links steht der Name, rechts steht der Text. Nur der Text rechts vom Doppelpunkt wird übersetzt. Der linke Teil bleibt gleich.

Richtig:
"save": "Speichern"
"cancel": "Abbrechen"
"open": "Öffnen"

Falsch:
"save": "save"
"Speichern": "Abbrechen"

Links bleibt der Name gleich. Rechts steht die Übersetzung. So kann das Programm die Datei richtig lesen.

Schritt 3: Auch eine KI darf helfen

Du kannst dir die Übersetzung auch von einer KI machen lassen. Kopiere den Inhalt der Sprachdatei in ChatGPT oder ein anderes KI-Werkzeug und sage dazu: Übersetze nur die Texte rechts, aber ändere die Namen links nicht.

Schritt 4: Speichern und neu starten

Speichere die Datei im Ordner lang und starte das Programm neu. Danach erscheint die neue Sprache automatisch in der Auswahl.

Teil 2: Ein eigenes Theme erstellen

Seit Version 4.1.1 werden Themes vollständig über externe JSON-Dateien im Ordner themes verwaltet. Du musst keine Python-Dateien mehr bearbeiten.

Schritt 1: Vorhandenes Theme kopieren

Öffne den Ordner themes und kopiere eine bestehende Datei, z.B. light.json. Benenne die Kopie um, z.B. my_theme.json.

Schritt 2: Farben anpassen

{
    "name": "my_theme",

    "bg": "#E0E0E0",              // Hintergrundfarbe der gesamten Anwendung
    "fg": "#000000",              // Hauptschriftfarbe
    "info_bg": "#ffffff",         // Textbereiche und Prozessprotokoll
    "separator": "#808080",       // Rahmen, Linien und Trennbereiche
    "entry_bg": "#ffffff",        // Eingabefelder und Kombinationsfelder
    "disabled_fg": "#6d6d6d",     // Deaktivierte Textelemente
    "error_bg": "#f2b8b5",        // Fehlerhervorhebungen
    "button_bg": "#d3d1ce",       // Schaltflächen und inaktive Tabs
    "button_hover": "#e4e0d8",    // Hover- und Aktiv-Zustand
    "text_relief": "sunken",      // flat, sunken, ridge oder groove
    "accent": "#0a64ad",          // Primäre Akzentfarbe
    "tree_selected": "#0a64ad",   // Hintergrund markierter Dateizeilen
    "tree_selected_fg": "#ffffff",// Textfarbe markierter Dateizeilen
    "tree_row_alt": "#f7f7f7"     // Farbe alternierender Zeilen
}

Speichern genügt. Das Programm erkennt die Datei automatisch.

Schritt 3: Theme auswählen

Starte das Programm. Alle gültigen JSON-Themes im Ordner themes erscheinen automatisch in den Optionen unter "App Theme".
